# Lucii-AI
Lucii Ai is a completely free unrestricted ai that can write you code, scripts, sniff networks and so much more.

stept to install below.

Install "unrestricted_ai_assistant3.0.py" (make sure to have python installed).

Once the tool has installed to your computer open your command prompt (windows).

In your command prompt CD into the directory you installed the tool to (example: cd C:\Users\chris\Desktop\Lucii AI).

Once your in the directory in your command prompt simply type "python unrestricted_ai_assistant3.0.py".

Once the tool has loaded click "Install Ollama" at the bottom of the tool (make sure to install the dolphin-llama3 version to ensure no restrictions).

Once Ollama is installed go back to the tool and click "Start Ollama" (this will automatically start Ollama for the tool).

Once installed and started the tool is ready to be used.
